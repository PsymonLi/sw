Radius

Remote Authentication Dial-In User Service is a networking protocol, operating on port 1812 that provides centralized Authentication, Authorization, and Accounting management for users who connect and use a network service.

The toggle allows for enabling/disabling Radius authentication.

Option Description
nas-id NasID is a string identifying the NAS(API Gw) originating the Access-RequestNasID is a string identifying the NAS(API Gw) originating the Access-Request
Server:Port <IP address>:<Port> of the RADIUS server
Server secret Secret is the shared secret between Venice and RADIUS server
auth-method AuthMethod is authentication method to use with the RADIUS serverAuthMethod is authentication method to use with the RADIUS server
Valid options include:
- pap
- peap-mschapv2
- eap-ttls-pap
