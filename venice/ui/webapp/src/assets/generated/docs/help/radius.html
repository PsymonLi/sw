Radius

Remote Authentication Dial-In User Service is a networking protocol, operating on port 1812 that provides centralized Authentication, Authorization, and Accounting management for users who connect and use a network service.

The toggle allows for enabling/disabling Radius authentication.

Option Description
enabled
nas-id NasID is a string identifying the NAS(API Gw) originating the Access-Request
url <IP address>:<Port> of the RADIUS server
secret Secret is the shared secret between API Gw and RADIUS server
auth-method AuthMethod is authentication method to use with the RADIUS server
trusted-certs TrustedCerts defines the set of PEM encoded root certificate authorities that will be used when verifying server certificates. It is. used in PEAP and EAP_TTLS auth methods
