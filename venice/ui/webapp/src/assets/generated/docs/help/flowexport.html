Flow Export

Venice allows to send specific flows to an external destination flow collector supporting IPFIX format. Users can send the flows to multiple destination flow collectors. To configure the flow export, the following are the configuration parameters.

Option Description
name Name of the object, unique within a Namespace for scoped objects. Should start and end in an alphanumeric character and can contain alphanumeric or .- characters. minimum length is 2 and maximum length is 64 charactersName of the object, unique within a Namespace for scoped objects. interval Interval defines how often to push the records to an external collector. The value is specified as a string format, '10s', '20m'Interval defines how often to push the records to an external collector.
